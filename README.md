# NutriRed

Plataforma digital para la gestión y redistribución de alimentos en Bogotá mediante un prototipo funcional interactivo.

## Objetivo
Optimizar la gestión de donaciones y reducir el desperdicio alimentario mediante herramientas digitales.

## Tecnologías utilizadas
- React
- TypeScript
- Vite
- Figma
- GitHub

## Metodología
- Scrum
- CDIO
